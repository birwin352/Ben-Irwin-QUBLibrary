# QUB Library Management System

This is a Java console application that manages a small library system.  
It allows users to add, remove, and display books and members, and handle borrowing and returning books.

## How to Run

Option 1 – Using the JAR file:
1. Open the project folder in a terminal.
2. Run:
   java -jar CSC1029Console.jar

Option 2 – From source code:
1. Open the project in a Java IDE (like IntelliJ or Eclipse), or
2. In a terminal, navigate to the `src` folder and compile:
   javac *.java
3. Then run:
   java Main

## Project Files
- src/ — contains all Java source files
- CSC1029Console.jar — executable version of the program
- ChatGPT_Report.docx — coursework documentation
- images/ — book covers for books stored in the library
