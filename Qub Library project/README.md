# QUB Library Console Application

This project is a Java console application developed for the CSC1029 module. It models a simple library system, including Books, Lending status, and user interaction through a menu.

2.  **Rename** this folder from **`Source files`** to **`src`**.

### Step 2: Set Up the Package Structure
